Instructions/Readme

(WINDOWS)
To run the decision tree learning algorithm (Assuming the JRE and JDK are installed):
    1. Run the command prompt.
    2. Use the "cd" command to get to the location of the source folder of Homework01, 
            cd ..\Homework01\src
            
    3. Use the command "javac Main.java" to compile the file "Main.java".
        This should create the class files for all the java files.
    4. Use the command "java Main" to run the program.
