        Windows and mac require the JRE and JDK to compile and run java files.

(WINDOWS/MAC)
To run this program:
    1. Run the command prompt (Windows) or run the terminal (Mac).
    2. Use the "cd" command to navigate to the location of the source folder in Homework02, 
            (Windows) --> cd "..\Homework02\src"
            (Mac) ------> cd "../Homework02/src"
            
    3. Use the command "javac KD_Builder.java" to compile the file "KD_Builder.java".
    4. Use the command "java KD_Builder 'inputfile' 'S'" to run the program.
            - 'inputfile' is the location of the file containing example inputs for the KDTree.
            - 'S' is the minimal set-size used by the algorithm to decide when to stop splitting data).
            
    
        NOTE: My program will read input files in the folder inputData, 
                at  "..\Homework02\src\inputData" (Windows)
                    "../Homework02/src/inputData" (Mac)
